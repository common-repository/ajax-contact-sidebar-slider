Instead of using a standard contact form template, you can download Contact Us Widget Slider in order to create unique contact forms with an assortment of available contact form layout options. Website visitor engagement is only successful if visitors can contact you easily, and Contact Us Widget Slider is the free way to collect relevant data from website visitors, while providing a simple method for visitors to contact your business.== Installation ==This section describes how to install the plugin and get it working.1. Install the wordpress plugin firstYou can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder2. Activate the plugin3. You will see "Ajax Contact Sidebar Slider" settings from right side of your wordpress menu.== Screenshots ==1. Screenshot 1== Changelog === 1.0 =* Stable Version Released
